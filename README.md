# Блогикум (часть 3)

Учебный проект блого‑платформы на Django из курса Яндекс Практикум.  
Цель проекта — отработать работу с моделями, представлениями (CBV/FBV), формами, аутентификацией, пагинацией и шаблонами.

## Возможности

- Публикации: создание/редактирование/удаление, отложенная публикация, черновики.
- Категории и локации (связанные сущности для постов).
- Комментарии к публикациям.
- Пользователи: регистрация/вход/выход, профиль пользователя.
- Админ‑панель для управления контентом.

## Технологии

- Python: рекомендуется 3.10 (совместимо с веткой Django 3.2)
- Django: 3.2.x
- БД: SQLite (по умолчанию)
- Верстка: `django-bootstrap5`

## Локальный запуск

### 1) Клонировать репозиторий и создать виртуальное окружение

```bash
git clone https://github.com/spiritbreaker/<repo>.git
cd <repo>/blogicum
python -m venv venv
```

Активировать окружение:

- Windows (PowerShell):
```bash
venv\Scripts\Activate.ps1
```

- Linux/macOS:
```bash
source venv/bin/activate
```

### 2) Установить зависимости и выполнить миграции

```bash
pip install -r requirements.txt
python manage.py migrate
```

### 3) (Опционально) Загрузить тестовые данные

В проекте есть фикстура `db.json`.

```bash
python manage.py loaddata db.json
```

### 4) Создать администратора и запустить сервер

```bash
python manage.py createsuperuser
python manage.py runserver
```

После запуска:
- сайт доступен по адресу: `http://127.0.0.1:8000/`
- админ‑панель: `http://127.0.0.1:8000/admin/`

## Тесты

Если в окружении установлен `pytest`:

```bash
pytest
```

Либо стандартные тесты Django (если нужны):

```bash
python manage.py test
```

## Структура проекта (кратко)

- `blog/` — основная логика блога (модели Post/Category/Location/Comment, страницы постов).
- `pages/` — статические страницы.
- `core/` — общие компоненты проекта.
- `templates/` — шаблоны.
- `static/` — статика.
- `db.json` — фикстура для заполнения БД.
